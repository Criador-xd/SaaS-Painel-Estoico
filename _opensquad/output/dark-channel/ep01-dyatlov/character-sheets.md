# 📋 INSTRUÇÕES DE USO

1. **Gere cada personagem separadamente** no Leonardo.ai usando os prompts acima.
2. **Salve cada imagem gerada** com o nome do personagem (ex: `CHAR-01-igor.png`).
3. **Use a imagem salva como "Image Reference"** no Leonardo.ai sempre que for gerar uma cena que inclui aquele personagem. Isso garante que o rosto e o corpo serão consistentes em todos os frames.
4. **Mantenha o mesmo preset/modelo** (ex: Nano Banana Pro) em TODAS as gerações para não quebrar o estilo visual.
5. **Strength do Image Reference:** Use entre 0.35 e 0.50 para manter a identidade sem travar demais a composição da cena.
