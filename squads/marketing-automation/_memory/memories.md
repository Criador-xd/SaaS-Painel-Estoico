# Memórias da Squad - Marketing Automation

## Versão Atual: v1.2 - Smart Analyzer

## O que faz:
1. **Analisa文件名** (nome do vídeo) para detectar tema
2. **Gera títulos otimizados** baseados em padrões de alto CTR
3. **Cria legendas persuasivas** com hooks emocionais
4. **Gera hashtags estratégicas** do nicho
5. **Cria CTA que converte** para YouTube e Instagram
6. **Aprova automaticamente** a publicação no SaaS
7. **Agenda para o próximo slot** (10-11h, 12-14h, 18-22h, 2-3h)

## Como funciona:
- Detecta tema: dinheiro, sucesso, empresário, espiritual, relacionamento
- Usa fórmulas de título que funcionam (ex: "O Segredo de X que Ninguém Te Conta")
- Adiciona power words e emojis para aumentar CTR
- Gera caption com hook → corpo → CTA → hashtags
- Publica em YouTube + Instagram simultaneamente

## Horários de publicação:
- Madrugada: 2h - 3h
- Manhã: 10h - 11h
- Tarde: 12h - 14h
- Noite: 18h - 22h
- Limite: 4 vídeos/dia, 28/semana
