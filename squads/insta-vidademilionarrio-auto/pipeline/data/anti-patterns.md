# Anti-Patterns: What to Avoid

## 1. The "Bro-Marketer" Cliché
- **Mistake**: Using aggressive, "fake" energetic language (e.g., "QUER FICAR RICO AGORA?!").
- **Why harmful**: Lowers perceived status. Elite brands don't shout; they attract.
- **Always do**: Use calm, authoritative, and selective words.

## 2. Platform fold ignorance
- **Mistake**: Placing the main hook after the first 125 characters in an Instagram caption.
- **Why harmful**: 80% of users will never click "...more" if the hook is hidden.
- **Always do**: Start with the hook using **Bold** or CAPITALIZED keywords.

## 3. Generic Visuals
- **Mistake**: Using stock photos of people in suits or fake cash piles.
- **Why harmful**: Screams "amateur" and "scam".
- **Always do**: Use high-end "B-Roll" (textures, architectural shots, luxury vehicles in shadow) or unique text-based designs.

## 4. CTA Ghosting
- **Mistake**: Finishing a great educational post with "Espero que ajude!".
- **Why harmful**: You lose the opportunity to start a sales conversation.
- **Always do**: Use a Keyword CTA (e.g., "Comente FOCO").

## 5. Over-complexity
- **Mistake**: Trying to explain 10 different tax strategies in one carousel.
- **Why harmful**: Attention spans are short. People save but never read complex content.
- **Always do**: One core insight per post. Deepen the value in the DM.
