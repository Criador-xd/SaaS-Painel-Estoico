# Domain Framework: Millionaire Branding & Instagram Mastery

1. **Strategic Diagnosis**
   - Identify the user's current awareness level (Unaware to Most Aware).
   - Select the psychological driver (Status, Freedom, or Control).

2. **Hook Engineering (The 1-2 Punch)**
   - Visual Hook: High-contrast luxury asset + polarizing statement.
   - Narrative Hook: The first line of the caption must create an "Open Loop" (unanswered question).

3. **Narrative Architecture**
   - **Educational (PAS)**: Present a common struggle -> Agitate the daily pain -> Introduce the Elite solution.
   - **Aspirational (BAB)**: Show the lifestyle (Before/Current) -> Pivot to the required mindset (After) -> Bridge with the CTA.

4. **Visual Language**
   - Use "Quiet Luxury" aesthetics: high negative space, serif typography, monochrome palettes + gold accents.
   - Slides must be scannable (max 20 words per slide).

5. **Conversion Optimization**
   - Transition from educational value to a specific "Keyword CTA" (e.g., "Comente MILIONARIO").
   - Guide the reader toward a DM interaction for high-ticket qualification.

6. **Feedback Loop**
   - Analyze engagement metrics (especially Shares and Saves).
   - Iteratively refine the hook style based on which patterns stop the scroll.
