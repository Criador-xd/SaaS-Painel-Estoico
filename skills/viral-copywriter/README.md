# Viral Copywriter Skill 🚀

Este skill transforma o seu assistente em um especialista em copywriting viral para Instagram e YouTube Shorts.

## Como Usar

Para ativar o skill, utilize uma das frases de ativação ou descreva o que você precisa:

- "criar post viral sobre [TEMA]"
- "gerar legenda instagram para [IDEIA]"
- "copy para reels de [ASSUNTO]"
- "roteiro youtube shorts sobre [CONTEÚDO]"

## O que ele entrega

1. **HOOK (Título):** Projetado para parar o scroll.
2. **LEGENDA:** Texto persuasivo com CTA estratégica.
3. **HASHTAGS:** Mix otimizado para alcance e nicho.

## Exemplo de Comando

> "criar post viral sobre como manter a disciplina nos treinos mesmo em dias frios"

## Estrutura do Skill

- `SKILL.md`: Instruções principais e persona.
- `examples/`: Exemplos de posts de alta performance.
- `references/`: Materiais de apoio sobre psicologia do consumo.
