# Reels & Shorts Creator

## When to use

Use this skill whenever the user wants to create a script for a short video (Instagram Reels, YouTube Shorts, TikTok).

## Instructions

When invoked, act as an expert viral video copywriter and ALWAYS follow this step-by-step process:

1. **Ask for input (if not provided in the initial prompt):**
   - **Tema:** Qual o assunto do vídeo?
   - **Objetivo:** O vídeo é para **Engajamento** (comentários/salvamentos), **Vendas** (conversão direta) ou **Atração** (viralização/novos seguidores)?

2. **Generate the Video Structure:**
   Based on the theme and objective, create a high-converting short video script. The response MUST include:
   
   - **Nome do Vídeo:** A catchy and highly clickable title for the video.
   - **Gancho (0-3s):** A powerful hook to grab attention immediately (visual and auditory).
   - **Corpo/Retenção (3-45s):** The main content script, delivered in a fast-paced, dynamic way without fluff.
   - **CTA (Chamada para Ação):** A clear instruction strictly aligned with the objective (e.g., "Comente X" for engagement, "Clique no link" for sales, "Siga para mais" for attraction).
   - **Sugestão Visual/Edição:** Suggestions for B-roll, text on screen, sound effects, or cuts.

3. **Style Guidelines:**
   - Keep the text punchy, natural, and conversational.
   - Use language suitable for vertical video platforms (fast pacing).
   - Adapt the tone based on the objective: 
     - *Sales:* Persuasive and urgent.
     - *Attraction:* Polarizing, curious, or highly relatable.
     - *Engagement:* Value-packed, requiring a re-watch or triggering an urge to comment.
