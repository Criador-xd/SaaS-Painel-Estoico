Crie um SaaS chamado "LifeFlow" - um gerenciador pessoal completo com 4 módulos: Tarefas, Finanças, Fitness e Bem-estar.

BACKEND (Supabase):

Crie as seguintes tabelas com colunas:

1. tarefas
   - id (UUID, primary key)
   - user_id (UUID, foreign key auth.users)
   - title (text)
   - priority (text: "baixa", "media", "alta")
   - done (boolean, default false)
   - due_date (date)
   - created_at (timestamp)

2. despesas
   - id (UUID, primary key)
   - user_id (UUID, foreign key auth.users)
   - amount (decimal)
   - category (text: "alimentacao", "transporte", "lazer", "saude", "outros")
   - description (text)
   - date (date)
   - created_at (timestamp)

3. fitness
   - id (UUID, primary key)
   - user_id (UUID, foreign key auth.users)
   - exercise_name (text)
   - completed (boolean, default false)
   - date (date)
   - created_at (timestamp)

4. wellness
   - id (UUID, primary key)
   - user_id (UUID, foreign key auth.users)
   - mood (integer 1-5)
   - note (text, opcional)
   - date (date)
   - created_at (timestamp)

Configure RLS (Row Level Security) em todas as tabelas para que cada usuário veja apenas os próprios dados.

PÁGINAS:

1. Login/Cadastro - Já incluso no Supabase Auth, use o template padrão do Lovable

2. Dashboard (página inicial após login)
   - Sidebar navegação à esquerda com ícones
   - 4 cards resumo: Total tarefas pendentes, Total despesas mês, Exercícios hoje, Humor médio
   - Links para cada seção

3. Tarefas
   - Formulário para adicionar tarefa: título, prioridade (dropdown), data
   - Lista de tarefas com checkbox para marcar como feito
   - Filtro: todas, pendentes, concluídas
   - Cor da prioridade: alta = vermelho, média = amarelo, baixa = verde

4. Finanças
   - Formulário: valor, categoria (dropdown), descrição, data
   - Lista de despesas do mês
   - Total por categoria (gráfico ou Cards)
   - Total geral do mês

5. Fitness
   - Lista de exercícios pré-definidos (alongamento, caminhada, treino, etc)
   - Checkbox para completar
   - Mostrar progresso do dia

6. Bem-estar
   - Selector de humor (1-5 emojis/carinhas)
   - Campo de anotação opcional
   - Histórico dos últimos 7 dias (gráfico de linha)

ESTILO RETRO/VINTAGE (mais leve e responsivo):

CORES:
- Fundo: #f5f0e6 (off-white/papel envelhecido)
- Texto principal: #2d2d2d (quase preto)
- Cor de acento: #8b4513 (marrom estilo papel kraft)
- Secundário: #d4c5a9 (bege/camurça)
- Sucesso: #4a7c59 (verde escuro/average)
- Alerta: #c9a227 (dourado/amarelo antigo)

TIPOGRAFIA:
- Título: "Courier New" ou "Playfair Display" (fonte máquina de escrever/serifada)
- Corpo: "Georgia" ou "Times New Roman" (serifada, leitura confortável)

ELEMENTOS VISUAIS:
- Bordas: linhas sólidas de 1-2px em marrom (#8b4513)
- Cards: bordas arredondadas sutis (8px), fundo levemente mais claro que o fundo principal
- Sombras: mínimas ou nenhuma (estilo papel)
- Ícones: simples, linha (outline style), não preenchidos
- Separadores: linhas horizontais discretas entre seções
- Buttons: bordas grossas (2px), efeito de pressão (pressionado ao clicar)

LAYOUT:
- Máximo 800px de largura centralizado
- Espaço generoso entre elementos (padding 24px+)
- Menu superior simples com logo + navegação
- Grid de 2 colunas no desktop, 1 coluna no mobile
- Mobile-first: touch-friendly, botões grandes (min 44px)

EFEITO RETRO:
- Sem gradientes modernos
- Sem sombras complexas
- Formulários com visual de caderno/papel
- Linhas zebradas sutil nas tabelas
